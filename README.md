# 个人代办事项应用

一个功能完善的纯前端代办事项管理应用。

## 功能特性
- ✅ 任务管理（添加、编辑、删除）
- ✅ 优先级和分类管理
- ✅ 截止日期提醒
- ✅ 数据导出/导入（JSON/CSV）
- ✅ 深色模式支持
- ✅ 实时搜索功能
- ✅ 统计和成就系统
- ✅ 键盘快捷键支持

## 技术特点
- 纯前端实现（HTML/CSS/JavaScript）
- 本地存储数据持久化
- 响应式设计，支持移动端
- 无任何外部依赖

## 本地运行
直接双击 todo-app.html 文件即可运行，或：
```bash
python3 -m http.server 8000
```
